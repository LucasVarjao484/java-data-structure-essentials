# Java Data Structure Essentials

Este repositório contém implementações de **estruturas de dados clássicas em Java**, desenvolvidas com foco em aprendizado, entendimento conceitual e construção manual das estruturas.

O objetivo **não é fornecer uma biblioteca pronta**, mas sim **demonstrar o raciocínio por trás de cada estrutura**, como eu consegui desenvolver suas operações e comportamentos durante a época de faculdade (2021-2024).

---

## Estruturas implementadas

- Vetor
- Lista encadeada
- Lista duplamente encadeada
- Lista duplamente encadeada com nó cabeça
- Lista circular
- Fila
- Pilha
- Deque
- Árvore binária
- Árvore de busca binária (BST)
- Árvore AVL

---

## Contexto e propósito

Este repositório reflete **diferentes momentos do meu aprendizado** em estruturas de dados.

Essas estruturas foram desenvolvidas durante disciplinas da faculdade, implementadas como atividades práticas, ou até escritas sob pressão de prova.

Por esse motivo, **nem todas representam a forma mais otimizada ou definitiva de resolver o problema**, mas todas representam o **meu entendimento real da lógica envolvida**.

Ainda possuo outras estruturas ainda da época de faculdade mais complexas (como árvores balanceadas, red-black, etc), mas estão sendo refatoradas e aprimoradas, com novos commits documentando essa evolução.
